# vim-setup

nvim and tmux config

## get tmux plugin manager

```bash
git clone https://github.com/tmux-plugins/tpm ~/.tmux/plugins/tpm
```

## Setup Requires

- [Nerd Font](https://www.nerdfonts.com/)

## reload

$ tmux source ~/.tmux.conf

## Install

```bash
brew install neovim
```

```bash
brew install tmux
```
